# CNN Image Classification

## Overview

This project implements an image classification model using Convolutional Neural Networks (CNN). The model is built with TensorFlow/Keras and is trained to classify grayscale images using deep learning techniques.

---

## Features

- Image Classification using CNN
- Data Preprocessing
- Image Augmentation
- Model Training
- Model Evaluation
- Prediction on New Images

---

## Technologies Used

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Google Colab

---

## CNN Architecture

- Conv2D
- MaxPooling2D
- Dropout
- Flatten
- Dense Layers
- Softmax Output

---

## Project Workflow

1. Load Image Dataset
2. Preprocess Images
3. Build CNN Model
4. Train the Model
5. Evaluate Accuracy
6. Predict Image Class
